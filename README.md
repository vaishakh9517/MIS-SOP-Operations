# MIS-and-Ops-Analytics-Using-Excel (Apex Motors Pvt. Ltd.)

This repository contains an end-to-end MIS Reporting, SOP Documentation, and Operations Tracking project designed to simulate real-world analytics and operations workflows in an automotive company.

The project demonstrates how raw transactional bike sales data from Apex Motors Pvt. Ltd. can be transformed into structured MIS reports, standardized SOPs, and actionable operations trackers to support business decision-making, operational control, and process improvement.
